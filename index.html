Features

  • Customizable file name, file type, and save directory
  • Captures screenshots in GIF, PNG, JPEG, or JPG formats
  • Automatically restarts for taking multiple screenshots

Make sstool a Startup App

Follow the steps below to make sstool a startup app on your PC:

  1. Step 1: Make a shortcut for the .exe file
  2. Step 2: Press Windows + R on your keyboard to open up the run menu
  3. Step 3: Type in "shell:startup" and press enter on your keyboard
  4. Step 4: Drag the shortcut that you made into the folder that opened when you hit enter
